## 社群行為準則

作為 CNCF 社群的貢獻者、維護者和參與者，為了促進開放且友善的社群，我們承諾尊重所有透過回報問題、發布功能需求、更新文件、提交 Pull Request 或修補程式、參加研討會或活動，以及參與其他社群或專案活動的貢獻者與參與者。

我們致力於讓所有人在參與 CNCF 社群時免受騷擾，不論其年齡、體型、種姓、身心障礙、族裔、經驗程度、家庭狀況、性別、性別認同與表達、婚姻狀況、軍人或退伍軍人身份、國籍、個人外貌、種族、宗教、性取向、社會經濟地位、部族，或任何其他多樣性維度。

## 範圍

本行為準則適用於：

* 在專案和社群空間內，
* 在其他空間中，當 CNCF 社群成員的個人言語或行為針對 CNCF 專案、CNCF 社群或 CNCF 活動中的其他 CNCF 社群成員時，也屬於此類情況。

## CNCF 活動

由 Linux 基金會與專業活動工作人員舉辦的 CNCF 活動，受 Linux 基金會[《活動行為準則》](https://events.linuxfoundation.org/code-of-conduct/)規定，該準則可於活動頁面取得。本準則設計為與 CNCF 行為準則搭配使用。

## 我們的標準

CNCF 社群是一個開放、包容且互相尊重的社群。我們社群的每位成員都有權讓自己的身份受到尊重。

有助於營造正向環境的行為包括但不限於：

* 對他人展現同理心與善意
* 尊重不同的意見、觀點和經驗
* 提供並從容接受建設性的回饋
* 承擔責任，向受我們錯誤影響的人道歉，並從經驗中學習
* 不只關注對個人有利的事，也著眼於整體社群的利益
* 使用友善且具包容性的語言

不可接受的行為範例包括但不限於：

* 使用帶有性意味的語言或圖像
* 挑釁、侮辱或貶低他人的言論，以及個人或政治攻擊
* 任何形式的公開或私下騷擾
* 未經明確許可，發布他人的私人資訊，例如實體地址或電子郵件地址
* 暴力行為、威脅暴力，或鼓勵他人參與暴力行為
* 未經當事人同意跟蹤或騷擾他人
* 未經同意的身體接觸
* 未經同意的性騷擾或求愛行為
* 將 CNCF 專案或社群空間用於與推動雲端原生技術無關的政治宣傳或政治立場推廣。需說明的是，本政策不限制個人穿著，包括表達個人信仰或身份認同的服裝。
* 其他在專業環境中被合理認為不適當的行為

同樣禁止以下行為：

* 在行為準則調查中故意提供虛假或誤導性資訊，或以其他方式故意干預調查。
* 因某人舉報事件或以證人身份提供事件相關資訊而對其進行報復。

專案維護者有權且有責任移除、編輯或拒絕不符合本行為準則的評論、提交、程式碼、Wiki 編輯、議題及其他貢獻。採用本行為準則即表示，專案維護者承諾公平且一致地將這些原則應用於管理 CNCF 專案的每個層面。不遵守或不執行行為準則的專案維護者可能被暫時或永久移除出專案團隊。

## 回報

對於在 Kubernetes 社群中發生的事件，請透過 [conduct@kubernetes.io](mailto:conduct@kubernetes.io) 聯絡 [Kubernetes 行為準則委員會](https://git.k8s.io/community/committee-code-of-conduct)。預計將在三個工作天內收到回覆。

對於其他專案，或不限於特定專案、影響多個 CNCF 專案的事件，請透過 [conduct@cncf.io](mailto:conduct@cncf.io) 聯絡 [CNCF 行為準則委員會](https://www.cncf.io/conduct/committee/)。或者，您可以聯絡 [CNCF 行為準則委員會](https://www.cncf.io/conduct/committee/)的任何個別成員提交報告。有關如何提交報告（包括如何匿名提交報告）的詳細說明，請參閱我們的[事件解決程序](https://github.com/cncf/foundation/blob/main/code-of-conduct/coc-incident-resolution-procedures.md)。預計將在三個工作天內收到回覆。

對於在 Linux 基金會舉辦的 CNCF 活動中發生的事件，請聯絡 [eventconduct@cncf.io](mailto:eventconduct@cncf.io)。

## 常見問題

如需進一步瞭解本行為準則，請參閱 [CNCF 行為準則常見問題](https://www.cncf.io/conduct/faq/)。

## 執法

在審查並調查所回報的事件後，具有管轄權的行為準則回應團隊將根據本行為準則及其相關文件決定採取何種適當行動。

有關哪些行為準則事件由專案負責人處理、哪些事件由 CNCF 行為準則委員會處理，以及哪些事件由 Linux 基金會（包括其活動團隊）處理的資訊，請參閱我們的[管轄權政策](https://github.com/cncf/foundation/blob/main/code-of-conduct/coc-committee-jurisdiction-policy.md)。

## 修訂

依據 CNCF 章程，對本行為準則的任何實質性修改必須獲得技術監督委員會的批准。
